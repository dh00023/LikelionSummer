## 해커톤

7월 24일 월요일까지 각자 맡은 부분 Gem이나 적용시켜오기

1. 기획
	- 그림
		- 색칠한 부분은 나만 볼 수 있어야한다.
		- 이미지로 부분을 클릭해서 팔, 다리 이런식으로 `z-index`이용해 쌓고, 선택한 이미지만 변경되도록한다.
	- 게시판
		- 세부사항 기록 게시판
		- 다른사람들과 내용 공유하는 게시판
		- 두 게시판은 클릭을 하면 변경되어야한다.
		- TinyMCE사용
	- 통계차트 자료
		- 어떤 부분이 가장 많이 선택되는지 차트로 보여준다.
2. 필요한 기능
	- **성인인증**
	- **성감대 표시하기**(송지연, 김선홍)
	- 게시판(이현경, 김승하)
	- 로그인(신명상, 김지훈)
	- 차트(정다혜)
